Main System Usage
How to's

The Dashboard Tab

The Dashboard is the main landing page when you enter the system. It's designed to give you a quick summary of all the important elements—trainings, trainees, and trainers—so you can get an overview at a glance.

Bell Icon (Notifications)

At the top right of the page, there is a bell icon which represents your notifications. The number next to it shows how many trainees are close to their certificate expiry date.

Clicking the bell opens a custom modal, and inside this modal, you will see a list of healthcare centres that currently have trainees nearing expiry. Each healthcare centre in the list can be expanded, and once you expand it, you'll see the list of individual trainees under that healthcare centre along with their details.

Trainings Section

This is the first section in the Dashboard. It displays four cards, each with a count:

  • Total Trainings – shows how many trainings exist in the system, regardless of status.
  • In Progress
  • Completed
  • Cancelled
  • Rescheduled

These represent the different statuses of trainings in the system. It's a quick summary so you can understand the training flow in one look.

Recent Trainings

Here, you will always see the 10 most recent trainings created or updated. For each training, the following details are shown:

  • Training Name
  • Trainer
  • Healthcare Centre
  • Start Date & Time
  • End Date & Time
  • Device Serial Number
  • Training Type
  • Training Status

This section helps you track recent activities and upcoming or ongoing trainings.

Trainees Section

There are three cards:

  • Total Trainees – shows how many trainees are in the system overall.
  • Active Trainees – trainees currently marked with status 'Active'.
  • Inactive Trainees – those not currently participating or active in training.

Trainers Section

Each card here represents one trainer. On the card, you'll see:

  • The trainer's name
  • Number of completed trainings
  • Number of trainings currently in progress

This gives a quick view of how active each trainer is.

The Trainee Database Tab

This page is where all your trainee data is stored and managed.

Register a Trainee Button

Right at the top, you'll find a Register a Trainee button. When clicked, it opens a form for registering a new trainee.

Search Bar

There's a universal search bar that allows you to search trainees using any of the following:

  • Name
  • IC Number
  • ID Number
  • Healthcare Centre
  • Status (active, inactive, or empty)

View Modes

You can choose how you want to view the data: Card View or Table View.

Card View:

  • Each card represents one trainee.
  • The title of the card is the trainee's name.
  • Quick info shown includes: Name, ID, IC/Passport, Healthcare, Latest Training, Re-certification Date, Completed Trainings, Status

When you click the card, a modal pop-up will appear showing detailed trainee information including training history.

In the modal, you'll find:

  • A Cancel button to close the modal.
  • An Edit button to modify certain fields
  • Editable: Name, IC/Passport, Email, Handphone Number, Healthcare, Designation, Area of Specialization, Serial Number, Trainee Status
  • Uneditable: Fields that are auto-calculated or used as system references.

Table View:

  • Each row is one trainee.
  • Click on any row to open the same detailed modal as in the card view.

The Trainee Management Tab

This page is split into two tabs:

  • All Trainings
  • Participating Trainees

Create a Training

At the top, there is a long button that says Create a Training. Clicking it brings you to a training creation form.

Search & Filters (All Trainings Tab only)

  • Search Bar: Search for any field.
  • Filter Section: Allows you to filter by Training ID, Trainer, Healthcare Centre, Device Serial Number, Training Type, Training Status
  • Click Apply Filters to search.
  • Click Reset Filters to clear everything.

Note: The filter section is not available in the Participating Trainees tab; only the search bar is present.

All Trainings Tab

Displays all trainings in a table view. Each row is one training, and the headers include:

  • Timestamp
  • Training ID
  • Trainer
  • Healthcare Centre
  • Start Date & Time
  • End Date & Time
  • Device Serial Number
  • Training Type
  • Gradebook Link
  • Status
  • WhatsApp Group
  • Actions

You can edit training information using the Actions > Edit button. This opens a modal where you can edit various fields and save changes.

Participating Trainees Tab

This tab shows cards, each representing one training. Each card displays Training ID, Gradebook Link, and Number of trainees.

Click View Trainees to open a modal where you can see detailed trainee information including performance descriptors and remarks.

The Questions Tab

This section manages your training assessment questions.

There are two tabs:

  • Main Training Questions
  • Refreshment Training Questions

Add Question

Each tab has a button:

  • "Add Main Training Question"
  • "Add Refreshment Training Question"

Question Cards

Each card shows:

  • The question as the title.
  • Options A to D (correct one in green).
  • The objective labeled below the options: Objective: [text].

Search Bar

Each tab has its own search bar for finding specific questions in that set.

The Settings Tab

This is where you can manage various system-wide settings.

Normal Users

Can only view current settings:

  • Healthcare Name
  • Device Serial Number
  • Area of Specialization
  • K-Laser Model

Admins Only

Have full control:

Add Setting Steps
  1. Choose the setting type (e.g., Area of Specialization)
  2. Enter the value
  3. Click Add Setting
Edit Setting Steps
  1. Click the Edit icon on a setting card
  2. A modal will appear
  3. Change the value and click Save
Delete Setting Steps
  1. Click the Trash icon
  2. Confirm the deletion in the alert

How to register a Trainee into the System

Registration Steps
  1. Navigate to the Trainee Database tab
  2. Click the "Register a Trainee" button at the top of the page
  3. Fill in all required trainee information in the registration form
  4. Verify all details are correct
  5. Click "Submit" to register the trainee
  6. The new trainee will appear in your trainee database

How to check the Trainee Details and modify trainee data

Check & Modify Steps
  1. Go to the Trainee Database tab
  2. Use the search bar to find the specific trainee or browse through the list
  3. Click on the trainee's card (Card View) or row (Table View)
  4. A detailed modal will open showing all trainee information
  5. To modify data, click the "Edit" button in the modal
  6. Update the editable fields as needed
  7. Click "Save Changes" to confirm modifications

How to create a training

Training Creation Steps
  1. Navigate to the Trainee Management tab
  2. Click the "Create a Training" button
  3. Fill in the training creation form with all required details
  4. Select the trainer, healthcare centre, and training type
  5. Set the start and end dates and times
  6. Assign the device serial number
  7. Add any additional training information
  8. Click "Create Training" to finalize

How to add a whatsapp group link

WhatsApp Link Steps
  1. Go to the Trainee Management tab
  2. Find the training in the "All Trainings" section
  3. Click the "Edit" button in the Actions column
  4. In the edit modal, locate the "WhatsApp Group" field
  5. Paste or type the WhatsApp group link
  6. Click "Save Changes" to update the training

How to modify training data

Modify Training Steps
  1. Navigate to Trainee Management > "All Trainings" tab
  2. Locate the training you want to modify
  3. Click the "Edit" button in the Actions column
  4. A modal will open with editable fields including: Trainer, Healthcare Centre, Start/End Date & Time, Device Serial, WhatsApp Group, Status
  5. Make the necessary changes
  6. Click "Save Changes" to confirm modifications

How to go to the gradebook

Gradebook Access Steps
  1. Go to the Trainee Management tab
  2. In the "All Trainings" section, find your training
  3. Click on the "Gradebook Link" in the training row
  4. This will open the gradebook for that specific training
  5. Alternatively, you can access gradebooks through the "Participating Trainees" tab by clicking the gradebook link on training cards

How to generate Training Materials

Generate Materials Steps
  1. Access the training you want to generate materials for
  2. Navigate to the training's gradebook
  3. Look for the "Generate Materials" or "Training Materials" button
  4. Click the button to generate the training materials
  5. The system will compile all necessary training documents and resources
  6. Download or access the generated materials as needed

How to get the Post test and Pre Test links

Get Test Links Steps
  1. Navigate to the specific training in Trainee Management
  2. Access the training's gradebook
  3. Look for the "Pre-Test" and "Post-Test" sections
  4. Click on "Generate Link" or "Get Link" for each test
  5. Copy the generated links
  6. Share these links with trainees for assessment purposes

How to Add or Remove Trainee's from Gradebook

Add Trainees Steps
  1. Open the training's gradebook
  2. Look for an "Add Trainee" or "Manage Participants" button
  3. Select trainees from the available list
  4. Click "Add Selected" to include them in the training
Remove Trainees Steps
  1. In the gradebook, find the trainee you want to remove
  2. Click the "Remove" or "Delete" button next to their name
  3. Confirm the removal when prompted
  4. The trainee will be removed from the training gradebook

How to Grade Hands on Marks

Grading Steps
  1. Access the training's gradebook
  2. Navigate to the "Hands-On Evaluation" section
  3. Find the trainee you want to grade
  4. Enter the hands-on performance score
  5. Add any remarks or comments about the trainee's performance
  6. Select the appropriate performance descriptor
  7. Click "Save Grade" to record the assessment

How to finish a training

Finish Training Steps
  1. Ensure all trainees have completed their assessments
  2. Verify all grades have been entered in the gradebook
  3. Go to the training in Trainee Management
  4. Click "Edit" on the training
  5. Change the training status to "Completed"
  6. Click "Save Changes"
  7. The training will be marked as finished and certificates can be generated

How to get the training package

Get Training Package Steps
  1. Navigate to the completed training
  2. Access the training's gradebook
  3. Look for a "Download Training Package" or "Export Package" button
  4. Click the button to generate the complete training package
  5. The package will include certificates, assessment results, and training materials
  6. Download the package to your device
  7. The package can be used for record-keeping and trainee documentation

How to add new Question

Add Question Steps
  1. Navigate to the Questions tab
  2. Choose between "Main Training Questions" or "Refreshment Training Questions" tab
  3. Click the appropriate "Add Question" button
  4. Enter the question text
  5. Add four answer options (A, B, C, D)
  6. Select the correct answer
  7. Specify the learning objective for the question
  8. Click "Save Question" to add it to the question bank

1. Pre-Training Test Results

Grade Description Trainer Focus
Outstanding Ready for advanced learning and leadership roles. Provide complex challenges and mentoring opportunities.
Above Average Minor knowledge gaps. Use examples and hands-on practice to reinforce confidence.
Average Basic knowledge but uncertain application. Emphasize fundamentals and regular feedback.
Below Average Struggles with core concepts. Provide one-on-one or small group sessions; use visuals.
Needs Improvement Lacks foundational knowledge. Start from basics; use interactive demonstrations; track progress closely.

2. Hands-On Evaluation

Grade Skills Demonstrated Conclusion
Outstanding Accurate application, confident supervision, protocol independence, and excellent safety compliance. Fully competent; ready to work independently.
Above Average Minor errors; solid grasp of protocols; performs well under minimal supervision. Needs light guidance to polish skills.
Average Understands basics but needs help with consistency, safety, and confidence. Requires more training and close supervision.
Below Average Struggles with key concepts and techniques. Needs significant retraining before independent practice.
Needs Improvement Unable to perform basic tasks or ensure safety. Comprehensive retraining required before advancement.

3. Post-Training Test Results

Grade Skill Set Evaluation Conclusion
Outstanding Mastery in all areas including photobiomodulation, laser parameters, and treatment techniques. Fully confident and capable of leading or training others.
Above Average Solid in most areas with minor gaps in complex applications. Ready for independent work with slight coaching.
Average Basic understanding; needs improvement in precision and consistency. Needs structured practice and supervision.
Below Average Inconsistent application and safety risks. Focused retraining and hands-on practice needed.
Needs Improvement No foundational competence in any area. Restart training from basics to develop competence.
